# Rushmore POC datasets

Generated datasets for the first proof of concept. Raw downloaded sources live in ignored `data/source/`; generated audit derivatives live in ignored `data/generated/`. App-facing JSON lives in this directory.

- `beverages.playable.json`: USDA FoodData Central April 2026 branded beverage dataset, deduped for gameplay. Includes `defaultPlayable` for cleaner prompts.
- `presidents.app-2026-06-22.json`: US presidents dataset with APP approval, executive-order, and election stats plus static identity metadata.
- `manifest.json`: counts, stat keys, source links, and sample top-four answers.
